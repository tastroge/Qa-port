Análise de Qualidade e Experiência do Usuário – Login

Pontos positivos:
- Interface simples
- Fluxo direto para autenticação

Pontos de melhoria:
- Mensagens de erro pouco claras
- Falta de feedback visual durante carregamento
- Não há orientação para recuperação de senha após erro

Sugestões:
- Exibir mensagens mais descritivas
- Adicionar indicador de carregamento
- Inserir link de recuperação de senha visível
